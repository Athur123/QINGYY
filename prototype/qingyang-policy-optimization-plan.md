# 青阳云SaaS社保公积金政策库页面优化计划

## 项目概述
优化青阳云HR SaaS平台中的社保公积金政策库管理页面，提升UI/UX体验。

## 设计系统
- **主色**: #5b8def（企业级蓝色）
- **背景**: 微暖灰白（#fafbfc、#f5f6f8）
- **阴影**: 柔和多层阴影系统
- **圆角**: 4-12px微圆角
- **动画**: 200ms过渡动画
- **字体**: Inter, Plus Jakarta Sans, JetBrains Mono（等宽数字）

## 技术实现

### CSS变量系统
```css
:root {
    /* 主色调 */
    --primary-500: #5b8def;
    --primary-50: #f0f5ff;

    /* 背景色 */
    --bg-primary: #ffffff;
    --bg-secondary: #fafbfc;
    --bg-tertiary: #f5f6f8;

    /* 文字色 */
    --text-primary: #1a1a1a;
    --text-secondary: #4a5568;
    --text-tertiary: #718096;

    /* 边框和阴影 */
    --border-light: #e8ecf1;
    --shadow-card: 0 2px 8px rgba(0, 0, 0, 0.04);
    --shadow-lg: 0 10px 15px -3px rgba(0, 0, 0, 0.05);

    /* 动画 */
    --transition-fast: 150ms ease;
    --transition-normal: 200ms ease;
}
```

### 响应式断点
- **1400px+**: 完整布局，浮动目录显示
- **1200px-1400px**: 目录宽度缩小
- **992px-1200px**: 目录隐藏，左侧导航树保持
- **768px-992px**: 单列布局
- **<768px**: 移动端适配
