# 青阳云HRO维度切换

青阳云系统有两个维度：EHR（核心人事）和HRO（外包派遣）。登录后默认在EHR维度，本技能用于切换到HRO维度（客户组织）。

## 切换流程

**第1步：确认当前页面**

用 `browser_snapshot` 获取页面快照，判断是否已在青阳云系统内。若页面不是青阳云系统，告知用户"请先登录青阳云系统"，停止执行。

**第2步：判断当前维度**

检查快照中是否存在HRO维度的页面特征（如顶部导航包含"上下游管理"字样）。若已确认在HRO维度，告知用户"当前已在HRO维度，无需切换"，停止执行。

**第3步：点击右上角头像**

用 `browser_click` 点击页面右上角的用户头像或"管理员"文字按钮，打开用户下拉菜单。若出现“切换内部组织”文字按钮，告知用户"当前已在HRO维度，无需切换"，停止执行。

**第4步：等待菜单展开**

用 `browser_wait_for` 等待"切换客户组织"文字出现（最多等 3 秒）。

**第5步：点击"切换客户组织"**

用 `browser_click` 点击下拉菜单中的"切换客户组织"选项。

**第6步：确认切换成功**

用 `browser_wait_for` 等待"上下游管理"文字出现（最多等 5 秒），确认顶部导航已切换为 HRO 维度。成功后告知用户切换完成。

## 异常处理

- 点击头像后若"客户组织"3秒内未出现，重试点击一次；仍失败则用 `browser_snapshot` 查看页面状态，告知用户"无法打开用户菜单，请手动操作"
- 点击"客户组织"后若页面未发生变化，告知用户"切换可能失败，请检查网络或手动切换"
- 切换成功后不要关闭浏览器
