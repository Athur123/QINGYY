# 员工变更记录查看功能设计

## Context

在员工详情页中增加"变更记录"入口，点击后从右侧滑出抽屉，展示该员工的字段级变更历史。

## 入口

员工详情页有"变更记录"按钮，点击打开变更记录抽屉。

## 变更记录抽屉布局

### 顶部
- 员工姓名 + 时间范围筛选（开始日期 ~ 结束日期）

### 变更时间线
按时间倒序排列，每条变更记录：

```
● 2026-04-21 10:30  修改了 姓名、部门
  ↓ 展开
    姓名: 张三 → 李四
    部门: 销售部 → 市场部

● 2026-04-20 14:20  新增了 手机
  ↓ 展开
    手机: - → 138xxxx

● 2026-04-19 09:15  删除了 备注
  ↓ 展开
    备注: 重要客户 → -
```

### 交互
- 默认收起，点击"展开"显示字段级变更
- 再点击"收起"
- 变更类型标签：新增（绿色）、修改（蓝色）、删除（红色）
- 旧值红色删除线，新值绿色高亮

## 数据结构

```javascript
{
  employeeId: "001",
  employeeName: "张三",
  changes: [
    {
      changedAt: "2026-04-21 10:30:00",
      changedBy: "操作人",
      changeType: "修改", // "新增" | "修改" | "删除"
      fields: [
        { field: "姓名", oldValue: "张三", newValue: "李四" },
        { field: "部门", oldValue: "销售部", newValue: "市场部" }
      ]
    }
    // ... more changes
  ]
}
```

## 文件

- `prototype/employee-detail.html` — 员工详情页（含变更记录入口和抽屉）
- 或修改现有员工详情页（如果有的话）

## 验证方式

1. 打开员工详情页
2. 点击"变更记录"按钮
3. 验证右侧抽屉滑入，显示变更时间线
4. 点击展开/收起，验证字段级变更显示正确
5. 测试时间范围筛选
