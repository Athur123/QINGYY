# 员工详情页优化设计规范

## Context

优化青阳云 HRO 系统的员工详情页（PersonalInfo），当前页面存在以下问题：
- 左侧边栏 220px 过宽，占用信息展示空间
- 表格样式陈旧（4列th/td交替背景）
- 动态记录 20+ 条平铺，喧宾夺主
- 信息分组（基本信息/工作信息）视觉区分不明显
- 字段空值显示为空白，用户不知道哪些字段需采集

## 设计目标

**现代化、紧凑高效、信息层次清晰** — 卡片式分组 + 浮动摘要 + 纵向字段布局

## 设计方案

### 1. 左侧边栏优化

**现状**: 220px 宽，包含 logo + 导航菜单
**优化**: 收缩至 64px 图标模式，或保持现有布局

若收缩至图标模式：
- Logo 缩小为图标
- 菜单项只显示图标，hover 显示文字提示
- 节省 156px 空间给主内容区

### 2. Tab 导航

- 高度 36px，紧凑设计
- 选中态：蓝色下划线 (2px) + 文字 `#2563EB` 加粗
- 未选中态：灰色文字 `#64748B`
- 支持横向滚动，Tab 宽度由内容撑开
- 激活底部高亮条，不使用背景色区分

### 3. 字段卡片式分组

**卡片样式**:
```
┌─────────────────────────────────────────────┐
│  ◆ 基本信息                    [编辑图标]  │  ← 蓝色左边框 (3px)
├─────────────────────────────────────────────┤
│  姓名              曾强勇                   │
│  性别              男                       │
│  手机              18575504349              │
│  电子邮箱                                   │  ← 空值只留标签
│  ...                                       │
└─────────────────────────────────────────────┘
```

**字段布局规则**:
- 布局方式：纵向列表，标签 + 值分行显示
- 标签列：固定 100px，12px 灰色 (#64748B)，500 字重
- 值列：自适应，14px 黑色 (#1E293B)，500 字重
- 空值处理：只显示标签，值区域留空（不显示"待填写"等提示）
- 分隔线：1px #E2E8F0
- 内边距：12px 16px

**字段值样式**:
- 普通文本：黑色
- 手机/证件：脱敏处理（中间部分用 *）
- 状态值：文字 + 彩色徽章
  - 在职：`#d1fae5` 背景 + `#065f46` 文字 + ✓ 图标
  - 离职：`#fee2e2` 背景 + `#991b1b` 文字
- 链接/可点击：蓝色 `#2563EB` + hover 下划线

### 4. 右侧浮动摘要

**员工卡片**:
```
┌─────────────────────┐
│  👤 曾强勇          │  ← 头像 + 姓名
│  在职 · 正式        │  ← 状态 + 试用期状态
├─────────────────────┤
│  手机 18575504349   │
│  生日 1995-08-25    │  ← 核心信息
│  入职 2025-02-25   │
├─────────────────────┤
│  动态记录 (3)   ▼   │  ← 点击展开更多
│  · 转正 2026-01-26 │
│  · 入职 2025-02-25 │
│  · 转正 2025-03-04 │
└─────────────────────┘
```

**样式**:
- 宽度：260px 固定
- 卡片背景：白色 + 轻微阴影
- 圆角：8px
- 内边距：16px
- 头像：48px 圆形，蓝色背景 + 白色首字

**动态记录**:
- 默认收起，显示最近 3 条
- 显示数量 badge，如 "动态记录 (3)"
- 点击展开显示全部
- 每条记录：类型 + 日期

### 5. 分组策略

**基础信息 Tab**:
- 基本信息：姓名、性别、手机、邮箱、证件类型、证件号码...
- 联系信息：现居住址、紧急联系人...
- 扩展信息：民族、政治面貌、婚姻状况...

**工作信息 Tab**:
- 职位信息：员工类型、试用期状态、入职日期、司龄、在职状态...
- 合同信息：合同主体、合同期限、签署日期、合同状态...

**其他 Tab**:
- 银行卡信息、教育经历、工作经历、证书资质...
- 每个 Tab 内容独立，根据字段数量自动调整高度

### 6. 交互细节

| 交互 | 行为 |
|------|------|
| Tab 切换 | 淡入淡出 150ms |
| 编辑字段 | 点击编辑图标进入编辑模式 |
| 展开动态 | 点击"展开"显示全部记录 |
| 收起动态 | 点击"收起"恢复默认 |
| 字段 hover | 无特殊效果（避免视觉噪音） |
| 链接 hover | 蓝色 + 下划线 |

### 7. 响应式策略

| 断点 | 布局调整 |
|------|----------|
| >= 1440px | 右侧摘要保持 260px |
| 1024-1439px | 右侧摘要收窄至 220px |
| < 1024px | 右侧摘要隐藏，点击按钮唤起 |
