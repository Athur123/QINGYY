# 员工详情页优化设计

## Context

优化青阳云 HRO 系统的员工详情页（PersonalInfo），当前页面存在以下问题：
- 12 个 Tab 全部平铺，信息密度高
- 表格布局紧凑，缺乏视觉层次
- 右侧动态记录占空间
- 字段分组不清晰

## 设计目标

**紧凑高效，优化视觉层次** — 保持信息密度，优化视觉层次，表格卡片化

## 设计方案

### 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  ← 返回                    花名册详情                        │
├─────────────────────────────────────────────────────────────┤
│ [基础信息] [合同信息] [银行卡] [教育] [工作] ... [参保档案]  │
├───────────────────────────────────────────┬─────────────────┤
│                                           │ 员工卡片(缩小)   │
│  ┌─ 基本信息 ──────────────────────┐    │ 姓名: 曾强勇    │
│  │  姓名: 曾强勇    性别: 男        │    │ 手机: 185...    │
│  │  手机: 185...    邮箱: ...       │    │ 在职状态: 在职  │
│  │  ...                              │    ├─────────────────┤
│  └───────────────────────────────────┘    │ 动态记录(精简)  │
│                                           │ · 转正 2026-01  │
│  ┌─ 工作信息 ──────────────────────┐    │ · 入职 2025-02  │
│  │  员工类型: 退休返聘  入职日期:.. │    │ · ...           │
│  │  在职状态: 在职     司龄: 13月   │    │                 │
│  └───────────────────────────────────┘    │                 │
├───────────────────────────────────────────┴─────────────────┤
```

### 1. Tab 导航优化

- 选中态更明显（底部高亮条 + 文字加粗）
- 未选中态颜色区分（灰色）
- 支持横向滚动
- 保持横排样式

### 2. 表格卡片化

- 基本信息、工作信息各自用卡片包裹
- 卡片标题加粗，带左侧色条
- 优化行列间距（16px 提升到 20px）
- 标签列（th）灰色背景 (#F8FAFC)

### 3. 右侧边栏精简

- 员工卡片只显示核心信息（姓名、手机、在职状态）
- 动态记录只显示最近3条
- 可点击"查看更多"展开

### 4. 字段分组

保持当前结构（基础信息 Tab 包含 基本信息 + 工作信息两个表格），仅优化样式。

## 实施范围

- 优化 `prototype/employee-detail.html` 页面
- 沿用青阳云设计系统风格

## 验证方式

1. 打开优化后的员工详情页
2. 验证 Tab 导航样式和选中态
3. 验证表格卡片化效果
4. 验证右侧边栏精简效果
