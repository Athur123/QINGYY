# 社保二级户设计文档

## 1. 业务背景

对于劳务派遣/人力资源公司，需要建立二级社保户结构，用于区分不同用工单位的工伤保险比例。不同用工单位因行业风险等级不同，工伤保险费率也不同。

### 业务模型

```
参保规则（一级）
├── 参保主体（用工单位）
├── 参保地区
├── 险种配置...
└── 二级户（N个）
    ├── 二级户名称
    ├── 用工单位编码
    ├── 用工单位全称
    └── 工伤保险费率
```

**关系**：一个参保规则下可建多个二级户，每个二级户对应一个用工单位，一个用工单位只能对应一个二级户。

---

## 2. 数据模型

### 2.1 二级户实体

| 字段名 | 字段说明 | 类型 | 必填 | 备注 |
|--------|----------|------|------|------|
| id | 二级户ID | string | 是 | 系统生成 |
| parentRuleId | 所属参保规则ID | string | 是 | 关联参保规则 |
| name | 二级户名称 | string | 是 | 如"XX公司-普通工种" |
| companyCode | 用工单位编码 | string | 是 | - |
| companyName | 用工单位全称 | string | 是 | - |
| injuryRate | 工伤保险费率 | decimal | 是 | 百分比，如 0.5 |

## 3. UI 设计方案

### 3.1 整体方案

在参保规则编辑页面中，使用**独立 Tab** 管理二级户。

### 3.2 Tab 结构

```
参保规则编辑页
├── 基本信息 Tab
├── 参保险种 Tab
├── 规则配置 Tab
└── 二级户管理 Tab  ← 新增
```

### 3.3 二级户管理 Tab 布局

```
┌─────────────────────────────────────────────────────────────┐
│  二级户列表（共 N 个）                                        │
│  ┌───────────────────────────────────────────────────────┐  │
│  │ 二级户名称  │ 用工单位编码 │ 用工单位全称 │ 工伤费率  │ 操作│
│  ├───────────────────────────────────────────────────────┤  │
│  │ XX公司-普通工种 │ 10001   │ XX科技有限公司 │ 0.5%   │编辑 删除│
│  │ XX公司-高危工种 │ 10002   │ XX科技有限公司 │ 1.5%   │编辑 删除│
│  └───────────────────────────────────────────────────────┘  │
│                                                              │
│  [+ 新增二级户]                                              │
└─────────────────────────────────────────────────────────────┘
```

### 3.4 新增/编辑二级户弹窗

```
┌───────────────────────────────────────┐
│  新增二级户                      [X]  │
├───────────────────────────────────────┤
│  * 二级户名称：______________________  │
│  * 用工单位编码：____________________  │
│  * 用工单位全称：____________________  │
│  * 工伤保险费率：__________%           │
│                                       │
│                 [取消]  [确定]         │
└───────────────────────────────────────┘
```

### 3.5 字段说明

| 字段 | 说明 | 验证规则 |
|------|------|----------|
| 二级户名称 | 二级户的显示名称 | 必填，最大50字符 |
| 用工单位编码 | 用工单位的唯一编码 | 必填，最大30字符 |
| 用工单位全称 | 用工单位的完整名称 | 必填，最大100字符 |
| 工伤保险费率 | 工伤保险缴纳比例 | 必填，0~100之间，最多2位小数 |

## 4. 交互逻辑

### 4.1 新增二级户
1. 点击"新增二级户"按钮
2. 弹出新增弹窗
3. 填写表单字段
4. 点击"确定"保存，二级户列表刷新
5. 点击"取消"关闭弹窗，不保存

### 4.2 编辑二级户
1. 点击某行"编辑"按钮
2. 弹出编辑弹窗（数据预填）
3. 修改表单字段
4. 点击"确定"保存更新
5. 点击"取消"关闭弹窗，不保存

### 4.3 删除二级户
1. 点击某行"删除"按钮
2. 弹出确认提示："删除后不可恢复，是否确认？"
3. 确认后删除，列表刷新
4. 取消则不删除

### 4.4 数据保存
- 二级户数据随参保规则主页面"保存"按钮一并保存
- 支持在二级户 Tab 中新增/编辑/删除后，统一保存
