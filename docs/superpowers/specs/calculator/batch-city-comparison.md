# 企业选址成本批量对比分析 - 设计文档

> **目标**: 重新设计社保公积金计算器，支持全国300+城市、1000+方案的大规模批量对比分析

---

## 1. 项目背景

### 当前限制
- `CalculatorCompare` 最多支持5个方案对比
- 需要手动逐个添加方案
- 不适合企业选址决策场景

### 新需求
- 支持全国300+城市批量对比
- 生成1000+方案组合（城市 × 薪资档位 × 公积金比例）
- 提供数据分析和可视化报告
- 辅助HR/决策者进行企业选址成本分析

---

## 2. 使用场景

### 主要用户
- HR负责人：评估不同城市的人才成本
- 企业决策者：选址决策支持
- 薪酬专员：制定多城市薪酬策略

### 典型流程
1. 设定对比参数（城市列表、薪资档位、公积金比例）
2. 系统自动生成所有方案组合
3. 通过表格查看详细数据（支持排序筛选）
4. 查看分析报告（TOP10最低成本、最优区间建议）
5. 导出数据用于汇报

## 4. 数据模型

### 4.1 CostScenario (单个方案)
```typescript
interface CostScenario {
  id: string;                    // 唯一标识
  cityCode: string;              // 城市编码
  cityName: string;              // 城市名称
  province: string;              // 省份
  cityTier: 'tier1' | 'newTier1' | 'tier2' | 'tier3';
  
  salary: number;                // 薪资档位
  baseAmount: number;            // 缴费基数
  
  costs: {
    companySocial: number;       // 单位社保
    companyFund: number;         // 单位公积金
    companyTotal: number;        // 单位合计
    personalSocial: number;      // 个人社保
    personalFund: number;        // 个人公积金
    personalTotal: number;       // 个人合计
    grandTotal: number;          // 总成本
  };
  
  policy: {
    minBase: number;
    maxBase: number;
    fundRate: number;
    rates: Record<string, { company: number; personal: number }>;
  };
}
```

### 4.2 ComparisonConfig (对比配置)
```typescript
interface ComparisonConfig {
  cities: string[];              // 选中城市编码
  salaryLevels: number[];        // 薪资档位
  fundRates: number[];           // 公积金比例
}
```

## 5. 核心组件

### 5.1 VirtualTable (虚拟滚动表格)
- **功能**: 展示1000+行数据，只渲染视口内10-20行
- **列**: 城市 | 档位 | 单位成本 | 个人成本 | 总成本
- **排序**: 点击表头切换升序/降序
- **固定列**: 城市名称固定
- **行高**: 固定50px

### 5.2 AnalysisReport (分析报告)
- **TOP10最低成本**: 表格+节省百分比
- **最优成本区间**: 基于员工规模的档位推荐
- **成本分布**: 区间分布柱状图

### 5.3 FilterPanel (筛选面板)
- 省份多选（按大区分组）
- 城市级别筛选
- 成本区间滑块

## 6. 交互设计

### 6.1 排序交互
```
点击表头 → 首次升序 → 再次降序 → 第三次取消
```

### 6.2 虚拟滚动
- 总行高 = 行数 × 50px
- 视口高度 = 600px
- 可见行数 = 12行
- 缓冲区 = 上下各5行
- 实际渲染 = 22行

### 6.3 数据生成
```javascript
// 约6000条组合（300城市 × 5档位 × 4比例）
const scenarios = [];
for (const city of cities) {
  for (const salary of salaryLevels) {
    for (const fundRate of fundRates) {
      scenarios.push(calculateScenario(city, salary, fundRate));
    }
  }
}
```

## 7. 性能优化

- **纯前端虚拟滚动**: 所有数据加载到内存
- **排序筛选**: 前端实时计算
- **初始加载**: 预计算所有方案，避免运行时重复计算
- **内存占用**: 约1000-6000条数据，现代浏览器可接受

---

## 8. 分析报告内容

### 8.1 TOP10成本最低城市
- 城市名、省份、总成本
- 单位成本、个人成本
- 相比全国平均的节省百分比

### 8.2 最优成本区间建议
- 不同薪资档位的年度总成本
- 性价比评分（考虑招聘难度）
- 推荐档位及理由

### 8.3 成本分布概览
- 成本区间分布柱状图
- 按城市级别分组的平均成本对比
